# Sanjuk-Unreal

UE5 애니메이션 관련 튜토리얼, 가이드, 리서치 자료 모음

## 구조

```
Sanjuk-Unreal/
├── Monolith-MCP-Tutorial/       # Monolith MCP 튜토리얼 (10편)
│   └── UE 5.7 + Claude Code로 애니메이션 제어
├── runreal-MCP-Tutorial/        # runreal MCP 튜토리얼 (12편)
│   └── Python 기반 UE5 자동화
├── AnimNext-Migration-Guide/    # AnimNext 마이그레이션 가이드 (13편)
│   └── 기존 ABP → AnimNext 전환
├── UAF-Setup-Guide/             # UAF 셋업 가이드 (12편)
│   └── Universal Animation Framework 구축
├── Monolith-Local-Setup-Guide.md  # Monolith 로컬 전용 설치법
└── Unreal_Briefing.md           # UE 애니메이션 데일리 브리핑 시스템 설명
```

## 주요 주제

| 주제 | 설명 |
|------|------|
| **Monolith MCP** | UE 5.7용, 115개 애니메이션 액션으로 Claude Code에서 직접 UE 제어 |
| **runreal MCP** | Python 스크립팅 기반 UE5 자동화, Monolith와 병행 사용 가능 |
| **AnimNext** | 차세대 애니메이션 프레임워크, 기존 ABP에서 마이그레이션 |
| **UAF** | Universal Animation Framework 플러그인 셋업 및 통합 |
| **데일리 브리핑** | 10개 카테고리 UE 애니메이션 최신 정보 자동 수집 (Notion + Telegram) |
