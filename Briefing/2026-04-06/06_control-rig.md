# Control Rig / IK 최신 기능

> 2026-04-06 브리핑 | [목차로 돌아가기](00_INDEX.md)

---

## UE 5.6 — "역대 최대 애니메이션 툴 업데이트"

- **인에디터 모프 타겟 생성/스컬프팅**: 스켈레탈 메시 에디터에서 블렌드 셰이프 직접 편집
- **Control Rig Physics (Experimental)**: 캐릭터 리그에 프로시저럴 피직스 추가
- **Motion Trails 재설계**: 뷰포트에서 아크/스페이싱 직접 편집
- **Tween Tools 강화**: 핫키 간접 제어, Overshoot 모드, Time Offset 슬라이더
- **Curve Editor 리디자인**: 키프레임 속도/성능 향상
- **Rigging Workshop 무료 공개**: FK/IK, 스플라인, 디포머, 블렌드 셰이프, Python 도구 커버

---

## UE 5.7 — 워크플로 최적화

- **리팩토링된 Animation Mode**: 도구 최소화 패널, 뷰포트 공간 확보
- **통합 Constraint UX**: Space + Constraints + Snapper → 단일 인터페이스
- IK Retargeter, Selection Set 기능 강화
- Full Body IK 시스템 완전 문서화
