# 심화 브리핑 — UE5 애니메이션 기술 종합 리서치

Claude Code Max 플랜 내에서 추가 비용 없이 6개 카테고리를 웹서치하여 종합 브리핑을 생성합니다.
자동 briefing.py(일일 모니터링)와 병행하는 심화 분석용 명령어입니다.

## 인수

- `$ARGUMENTS` — 선택. 특정 주제(예: "AnimNext", "Motion Matching")를 지정하면 해당 주제 중심으로 심화. 없으면 전체 카테고리 브리핑.

## 실행 순서

### 1단계: 6개 카테고리 병렬 웹서치

아래 검색을 **병렬로** 수행합니다. 각 카테고리별 최소 2개 쿼리:

#### 카테고리 1: 공식 채널
- `site:unrealengine.com animation 2026` (Tech Blog, 릴리즈 노트)
- `site:dev.epicgames.com animation workflow 2026` (공식 문서)

#### 카테고리 2: 커뮤니티 동향
- `Unreal Engine animation Discord community news 2026`
- `Unreal Slackers animation discussion 2026`

#### 카테고리 3: 업계 뉴스
- `site:cgchannel.com Unreal Engine animation 2026`
- `site:awn.com Unreal Engine 2026`

#### 카테고리 4: GitHub 오픈소스
- `Unreal Engine 5 animation GitHub repository new 2026`
- `awesome-unreal animation tools plugin 2026`

#### 카테고리 5: 핵심 기술 키워드 (매번 필수)
다음 키워드를 최소 3개씩 묶어 검색:
- `UE5 AnimNext UAF update 2026`
- `UE5 Motion Matching PoseSearch Mover Plugin 2026`
- `UE5 Control Rig MetaHuman ML Deformer 2026`
- `UE5 Chaos Cloth physics simulation 2026`
- `Unreal Engine MCP AI automation Monolith 2026`
- `UE5 GASP Game Animation Sample 2026`

#### 카테고리 6: 교육 & 심화
- `Unreal Engine technical animation tutorial course 2026`
- `JALI VibeUE Motorica Cascadeur Unreal 2026`

### 2단계: 이전 브리핑과 비교

1. `Briefing/` 폴더에서 가장 최근 브리핑 날짜 확인
2. 이전 브리핑 대비 **신규 정보**에 🆕 표기
3. 변화 없는 항목은 간략히 "변경 없음"으로 표기

### 3단계: 브리핑 문서 생성

`Briefing/YYYY-MM-DD/` 폴더에 기존 구조와 동일하게 생성:

```
Briefing/YYYY-MM-DD/
├── 00_INDEX.md          — 목차 & 버전별 핵심 변화
├── 01_executive-summary.md  — 핵심 요약 & 트렌드
├── 02_animnext-uaf.md       — AnimNext / UAF
├── 03_motion-matching.md    — Motion Matching & PoseSearch
├── 04_metahuman.md          — MetaHuman Animator
├── 05_ai-animation.md       — AI 기반 애니메이션 도구
├── 06_control-rig.md        — Control Rig / IK
├── 07_mcp-ecosystem.md      — MCP 생태계 & AI-UE 연동
├── 08_community-tools.md    — 커뮤니티 도구 & GitHub
├── 09_action-items.md       — TA 액션 아이템 & 로드맵
```

### 4단계: 소스 명시

각 섹션 하단에 **Sources** 섹션 필수:
```
## 소스
- [제목](URL) — 날짜, 핵심 내용 한줄
```

### 5단계: 결과 요약

브리핑 완료 후 요약 테이블 출력:

| 카테고리 | 검색 수 | 신규 정보 | 주요 발견 |
|----------|---------|----------|----------|
| 공식 채널 | N | N건 | ... |
| 커뮤니티 | N | N건 | ... |
| ... | ... | ... | ... |

## 특정 주제 모드

`$ARGUMENTS`가 지정된 경우:
- 해당 주제에 대해 **심화 검색** (카테고리당 3-4개 쿼리)
- 관련 GitHub 리포 상세 분석
- 실무 적용 방안까지 포함
- 파일은 `Briefing/YYYY-MM-DD/deep-dive-{주제}.md`로 생성
