# 1. FocalRig이란?

## 1.1 FocalRig 소개

**FocalRig**은 Unreal Engine용 Control Rig 플러그인으로, **프로시저럴(절차적) Look & Aim 시스템**을 제공합니다. 쉽게 말해, 캐릭터가 특정 대상을 바라보거나 조준할 때 필요한 몸 전체의 자연스러운 움직임을 **자동으로** 만들어주는 도구입니다.

### 한줄 요약

> "Aim Offset 없이도, 단 하나의 포즈에서 픽셀 퍼펙트한 풀바디 에이밍을 구현하는 플러그인"

## 1.2 왜 FocalRig이 필요한가?

### 기존 방식의 문제점: Aim Offset

UE5에서 캐릭터가 위/아래/좌/우를 조준하려면, 전통적으로 **Aim Offset**을 사용합니다.

```
기존 Aim Offset 방식:

1. 각 방향별 포즈를 제작해야 함 (최소 9개: 상/중/하 × 좌/중/우)
2. 이 포즈들 사이를 블렌딩하여 중간 방향을 표현
3. 상반신에만 적용되는 경우가 많음 (하반신은 별도 처리)
4. 극단적 각도에서 부자연스러운 보간이 발생
```

#### Aim Offset의 단점

| 문제 | 설명 |
|------|------|
| 많은 에셋 필요 | 방향별 포즈를 하나하나 만들어야 함 |
| 상반신만 움직임 | 허리 아래는 고정 → 부자연스러움 |
| 블렌딩 아티팩트 | 극단적 각도에서 관절이 꺾이거나 떨림 |
| 스켈레톤 의존 | 새 캐릭터마다 다시 만들어야 함 |
| 유지보수 비용 | 포즈 변경 시 모든 방향을 다시 수정 |

### FocalRig의 해결 방식

```
FocalRig 방식:

1. 단 1개의 기본 포즈만 있으면 됨
2. 프로시저럴(절차적)로 풀바디 회전 계산
3. 스파인(척추)부터 골반까지 전신이 자연스럽게 회전
4. 다리까지 보정하여 발이 땅에서 미끄러지지 않음
5. 스켈레톤을 자동 분석하여 설정 → 어떤 캐릭터든 바로 적용
```

## 1.3 FocalRig vs Aim Offset 비교

| 항목 | Aim Offset | FocalRig |
|------|-----------|----------|
| 필요한 포즈 수 | 9개 이상 | **1개** |
| 적용 범위 | 상반신 위주 | **풀바디 (전신)** |
| 정확도 | 블렌딩 기반 (근사치) | **픽셀 퍼펙트** |
| 스켈레톤 호환성 | 스켈레톤별 별도 제작 | **자동 설정** |
| 세팅 시간 | 포즈 제작 포함 수 시간 | **수 분** |
| 다리 보정 | 미지원 (별도 구현 필요) | **내장 기능** |
| 눈 추적 | 미지원 | **내장 기능** |
| 반동 시스템 | 미지원 | **내장 기능** |
| 퍼포먼스 | 보통 | **100 마이크로초 미만** |

## 1.4 FocalRig의 핵심 기능 4가지

### 1) Aim Chain (에임 체인)

캐릭터의 **척추(Spine) 전체를 부드럽게 회전**시켜, 시점 본(Point-of-View Bone)이 타겟을 정확히 조준하게 합니다.

```
예시: 캐릭터가 오른쪽 위의 적을 조준할 때

[Aim Offset]              [FocalRig Aim Chain]
    ↓                           ↓
상반신만 틀어짐             전신이 자연스럽게 회전
허리는 고정                 척추가 아래서부터 순차 회전
다리는 그대로               골반도 살짝 회전
                            다리가 자동으로 보정됨
```

### 2) Eye Aim (아이 에임)

캐릭터의 **눈이 타겟을 자연스럽게 추적**합니다. 실제 사람처럼 **Saccade(급속 안구 운동)** — 시선이 고정점 사이에서 빠르게 튀는 미세한 무작위 움직임 — 을 시뮬레이션합니다.

```
특징:
- 스프링 댐핑 기반 부드러운 추적
- 리얼리스틱 Saccade (미세 진동)
- 각도 제한 설정 가능
- 최소 거리 설정 → 가까운 대상에 사팔뜨기 방지
```

### 3) Recoil (반동)

무기나 카메라의 **반동 패턴**을 작성/재생합니다. 2D 그리드 에디터에서 드래그 앤 드롭으로 패턴을 그릴 수 있습니다.

```
내장 프리셋:
├── SMG (기관단총)
├── Assault Rifle (돌격소총)
├── LMG (경기관총)
└── Shotgun (산탄총)
```

### 4) Leg Adjustment (다리 보정)

상반신이 회전할 때 **발이 바닥에 고정**되도록 보정합니다. 골반이 돌아가도 발이 미끄러지지 않습니다.

## 1.5 퍼포먼스

FocalRig의 가장 큰 장점 중 하나는 **극한의 성능**입니다.

```
캐릭터 리그 전체 실행 시간: 100 마이크로초 미만 (워커 스레드 기준)

비교:
- Full-Body IK 노드 1개 > FocalRig 전체 리그 실행
- 즉, FocalRig은 FBIK 하나보다도 가볍습니다
```

> 💡 100 마이크로초 = 0.1 밀리초 = 0.0001초. 60 FPS 기준 한 프레임이 16.67ms이므로, FocalRig은 프레임 시간의 **0.6%** 미만만 사용합니다.
