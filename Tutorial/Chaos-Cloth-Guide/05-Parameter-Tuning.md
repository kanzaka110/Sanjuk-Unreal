# 5. 파라미터 튜닝 — 원하는 소재감 만들기

## 5.1 파라미터가 필요한 이유

같은 천 시뮬레이션이라도 파라미터에 따라 **실크처럼 부드러울 수도, 가죽처럼 뻣뻣할 수도** 있습니다.

```
[실크 스카프]          [면 티셔츠]          [가죽 코트]
 ~~ ~~~ ~~             ┌──┐ ┌──┐           ┌──────┐
  ~~ ~~ ~~             │  └─┘  │           │      │
   ~~ ~~ ~             │      │           │      │
    부드럽게             적당히 접힘          뻣뻣하게
    흘러내림                                 서있음
```

## 5.2 어디서 설정하나?

### Clothing Tool 방식

```
SkeletalMesh 에디터 → Asset Details →
  └─ Clothing 섹션 → 생성한 Cloth 항목 펼치기 →
     └─ Cloth Config 섹션에서 설정
```

### Panel Cloth (Dataflow) 방식

```
Chaos Cloth Asset 에디터 →
  └─ SimulationSolveConfig 노드의 속성에서 설정
```

## 5.3 핵심 파라미터 — 하나씩 설명

### Stiffness (강성) — "얼마나 뻣뻣한가?"

#### Bend Stiffness (굽힘 강성)

천이 굽히는 것에 얼마나 저항하는지 결정합니다.

| 소재 | 값 범위 | 느낌 |
|------|--------|------|
| 실크, 쉬폰 | **0.01~0.05** | 흘러내리듯 부드러움 |
| 면, 리넨 | **0.1~0.5** | 일반 옷감 |
| 데님, 캔버스 | **1.0~5.0** | 뻣뻣한 청바지 느낌 |
| 가죽, 갑옷 천 | **5.0~10.0** | 딱딱하게 서있음 |

```
                    Bend Stiffness
        낮음 ◀─────────────────────▶ 높음

        ~~~~                      ┃
         ~~~~                     ┃
          ~~~~                    ┃
           ~~~~                   ┃
     실크처럼 흘러내림        가죽처럼 뻣뻣
```

> **Bending Elements** 옵션: 활성화하면 과도한 변형을 방지합니다. 대부분의 경우 켜두세요.

#### Stretch Stiffness (늘어남 강성)

천이 얼마나 늘어나는지 제어합니다.

- **높은 값** (대부분의 직물): 거의 늘어나지 않음
- **낮은 값** (스판덱스, 니트): 탄성 있게 늘어남

> **초보자 팁:** 대부분의 경우 **높은 값을 유지**하세요. 소재감은 Bend Stiffness로 표현합니다.

### Density (밀도) — "얼마나 무거운가?"

천의 질량을 결정하여 **중력에 얼마나 반응하는지** 정합니다.

| 소재 | Density | 효과 |
|------|---------|------|
| 가벼운 스카프 | 낮음 | 바람에 쉽게 날림, 느리게 떨어짐 |
| 일반 의류 | 중간 | 자연스러운 드레이핑 |
| 두꺼운 코트 | 높음 | 묵직하게 떨어짐, 바람에 잘 안 날림 |

```
        Density
  낮음 ◀───────▶ 높음

   ↗ 가볍게       ↓ 묵직하게
    날림           떨어짐
```

### Damping (감쇠) — "얼마나 빨리 멈추나?"

시뮬레이션의 진동(흔들림)을 억제합니다.

```
Damping 낮음 (0.01):  ~~~~~~~~~~~~~~~~~~~~~  오래 흔들림
Damping 중간 (0.1):   ~~~~~~------          적당히 멈춤
Damping 높음 (0.5):   ~~---                 빠르게 멈춤
```

| 값 | 느낌 | 적합한 소재 |
|----|------|-----------|
| 0.01~0.05 | 오래 펄럭임 | 깃발, 실크 스카프 |
| 0.1~0.3 | 적당히 안정 | **일반 의류 (추천 시작값)** |
| 0.3~0.5 | 빠르게 안정 | 무거운 커튼, 코트 |

> **너무 높으면** 천이 뻣뻣해 보이고, **너무 낮으면** 끝없이 흔들립니다.

## 5.4 바람/공기 파라미터

### Drag (항력) — "공기 저항"

천이 공기 중에서 이동할 때 받는 저항입니다.

```
캐릭터가 달릴 때:

  Drag 낮음:  망토가 가만히 있음  (공기 저항 없음)
  Drag 높음:  망토가 뒤로 펄럭!   (공기 저항 많음)
```

### Lift (양력) — "위로 뜨는 힘"

천이 공기 흐름에 의해 위로 들리는 효과입니다.

```
  Lift 높음:  망토가 위로 뜸  ↗
  Lift 낮음:  망토가 아래로   ↘
```

### 바람과 이동의 관계

```
[중요!] Drag/Lift는 두 가지에 반응합니다:

1. Wind Directional Source (환경 바람)
   └─ 레벨에 배치한 바람 소스

2. 캐릭터 이동 속도 (상대 바람)
   └─ 캐릭터가 달리면 자동으로 역방향 바람 효과
   └─ 바람 소스가 없어도 망토가 펄럭이는 이유!
```

### Wind 설정 (레벨)

```
1. Place Actors → "Wind Directional Source" 검색 → 레벨에 배치
2. Details 패널:
   └─ Speed: 바람 세기 (100~500 시작)
   └─ 방향: 화살표가 가리키는 쪽으로 바람이 붑니다
3. Cloth가 바람에 반응하는지 Simulate로 확인
```

## 5.5 솔버 파라미터 — 정확도 vs 성능

### Iteration Count (반복 횟수)

물리 연산을 몇 번 반복할지 정합니다. **많을수록 정확하지만 느립니다.**

| 값 | 정확도 | 성능 | 용도 |
|----|--------|------|------|
| 1~3 | 낮음 (관통 가능) | 빠름 | 모바일, 배경 NPC |
| **3~5** | **중간 (대부분 OK)** | **보통** | **게임플레이 (추천 시작값)** |
| 5~10 | 높음 | 느림 | 주인공, 중요 캐릭터 |
| 10~20 | 매우 높음 | 매우 느림 | 시네마틱 전용 |

### Substep (서브스텝)

한 프레임의 물리 연산을 여러 단계로 분할합니다.

```
Substep 1:  프레임당 1번 계산 (기본)
Substep 2:  프레임당 2번 계산 (빠른 움직임에 유용)
Substep 3:  프레임당 3번 계산 (정밀, 비용 높음)
```

| 상황 | 권장 Substep | 이유 |
|------|-------------|------|
| 일반 움직임 | 1 | 기본값 충분 |
| 빠른 달리기/전투 | 2~3 | 관통 방지 |
| 시네마틱 | 3~5 | 최고 정확도 |

### 비용 공식 (이해만 하면 OK)

```
총 비용 ≈ 파티클 수 × Iteration × Substep

예시:
  500파티클 × 5 Iteration × 1 Substep = 2,500 연산/프레임
  500파티클 × 5 Iteration × 3 Substep = 7,500 연산/프레임 (3배!)
```

> **Iteration과 Substep은 곱셈입니다!** 둘 다 올리면 비용이 급증합니다.

## 5.6 소재별 프리셋 — 복사해서 쓰세요

### 실크 / 쉬폰 (부드럽고 가벼움)

```
Bend Stiffness:    0.01~0.05
Stretch Stiffness: 높음
Density:           낮음
Damping:           0.01~0.05 (오래 흔들림)
Drag:              높음 (공기에 잘 반응)
Lift:              중간
Iteration:         5~8
```

### 면 / 리넨 (일반 의류) — 초보자 추천 시작값

```
Bend Stiffness:    0.1~0.5
Stretch Stiffness: 높음
Density:           중간
Damping:           0.1~0.3
Drag:              중간
Lift:              낮음
Iteration:         3~5
```

### 가죽 / 두꺼운 코트 (무겁고 뻣뻣)

```
Bend Stiffness:    2.0~5.0
Stretch Stiffness: 매우 높음
Density:           높음
Damping:           0.3~0.5 (빠르게 안정)
Drag:              낮음 (무거워서 바람 영향 적음)
Lift:              매우 낮음
Iteration:         3~5
```

### 망토 / 깃발 (펄럭이는 느낌)

```
Bend Stiffness:    0.05~0.2
Stretch Stiffness: 높음
Density:           중간~낮음
Damping:           0.05~0.1 (적당히 흔들림)
Drag:              높음 (펄럭임의 핵심!)
Lift:              높음
Iteration:         5~10
```

### 퍼프 스커트 / 볼가운 (부풀어오른 느낌)

```
Bend Stiffness:    1.0~3.0 (형태 유지)
Stretch Stiffness: 높음
Density:           중간
Damping:           0.2~0.3
Self Collision:    ON (필수 — 접힘 방지)
Iteration:         5~8
```

## 5.7 튜닝 워크플로우 — 초보자 순서

```
Step 1: 프리셋 중 가장 가까운 것을 선택
        └─ 위의 소재별 프리셋 참고
               ↓
Step 2: Density 먼저 조절
        └─ 시뮬레이션 → 중력 반응 확인
        └─ 너무 가벼우면 올리고, 무거우면 내리기
               ↓
Step 3: Bend Stiffness 조절
        └─ 소재감 결정 (부드러움 ↔ 뻣뻣함)
               ↓
Step 4: Damping 조절
        └─ 흔들림 멈추는 속도 확인
               ↓
Step 5: Drag/Lift 조절
        └─ 캐릭터 달리기 → 망토/치마 반응 확인
        └─ 바람 소스 추가 → 바람 반응 확인
               ↓
Step 6: Iteration/Substep 조절
        └─ 관통 발생 시에만 올리기
        └─ 성능 체크 필수 (stat Cloth)
               ↓
Step 7: 다양한 애니메이션으로 최종 확인
        └─ 걷기, 달리기, 점프, 180도 회전
```

## 5.8 자주 하는 실수

| 실수 | 증상 | 해결 |
|------|------|------|
| 모든 값을 한꺼번에 변경 | 어디가 문제인지 모름 | **한 번에 하나씩** 변경 |
| Iteration을 너무 높게 | 프레임 드롭 | 3~5에서 시작 |
| Drag/Lift를 너무 높게 | 비현실적으로 날림 | 0.5 이하로 시작 |
| Damping을 0으로 | 끝없이 흔들림 | 최소 0.05 이상 |
| Density만 높게 | 무겁기만 하고 뻣뻣하지 않음 | Bend Stiffness도 함께 올리기 |
