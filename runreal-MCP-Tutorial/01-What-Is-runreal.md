# 01. runreal이란?

## 한 줄 요약

**UE5 에디터에 플러그인 없이 AI를 연결하는 MCP 서버**

## MCP가 뭔가요?

MCP(Model Context Protocol)는 AI 모델(Claude, GPT 등)이 외부 도구를 사용할 수 있게 해주는 표준 프로토콜입니다.

```
사용자 → "레벨에 큐브 10개 배치해줘"
    ↓
Claude Code → MCP 도구 호출 (editor_create_object)
    ↓
runreal MCP 서버 → Python 코드 생성
    ↓
UE5 에디터 → Python 실행 → 큐브 10개 생성됨!
```

## 아키텍처

```
┌──────────────────┐
│  Claude Code     │  ← AI 어시스턴트
│  (MCP Client)    │
└────────┬─────────┘
         │ stdio (JSON-RPC)
         ▼
┌──────────────────┐
│  @runreal/       │  ← Node.js MCP 서버 (로컬에서 실행)
│  unreal-mcp      │
└────────┬─────────┘
         │ UDP 멀티캐스트 (에디터 찾기)
         │ + TCP (명령 전송)
         ▼
┌──────────────────┐
│  UE5 에디터      │  ← Python Remote Execution이 명령을 받아 실행
│  (Python Plugin) │
└──────────────────┘
```

### 연결 과정 상세

1. **디스커버리 (UDP)**: MCP 서버가 `239.0.0.1:6766`으로 "UE 에디터 어디있어?" 브로드캐스트
2. **응답 (UDP)**: UE 에디터가 "나 여기있어, TCP 포트는 6776이야" 응답
3. **연결 (TCP)**: MCP 서버가 `127.0.0.1:6776`으로 TCP 연결
4. **명령 실행**: Python 코드를 JSON으로 감싸서 전송 → 결과 반환

## 다른 UE5 MCP와 뭐가 다른가요?

| 비교 항목 | **runreal** | **Monolith** | **chongdashu** | **flopperam** |
|----------|:-----------:|:------------:|:--------------:|:-------------:|
| UE 플러그인 설치 | **불필요** | 필요 (C++) | 필요 (C++) | 필요 (C++) |
| 설치 시간 | **30초** | 10분+ | 10분+ | 10분+ |
| 프로젝트 파일 수정 | **없음** | 있음 | 있음 | 있음 |
| P4에 영향 | **없음** | 관리 필요 | 관리 필요 | 관리 필요 |
| 전용 도구 수 | 19개 | 815개 | 20개+ | 40개+ |
| 애니메이션 전용 도구 | 없음 (Python으로 대체) | **115개** | 없음 | 11개 |
| Python API 전체 접근 | **가능** | 불가 | 제한적 | 제한적 |
| 확장 용이성 | **Python만 추가** | C++ 컴파일 필요 | C++ 필요 | C++ 필요 |
| UE 최소 버전 | **5.4** | 5.7 | 5.5 | 5.5 |

## runreal을 선택해야 하는 경우

- 회사 P4에 플러그인을 올릴 수 없을 때
- UE 5.4~5.6을 사용할 때 (Monolith는 5.7+ 필수)
- Python 스크립팅으로 자유롭게 확장하고 싶을 때
- 빠르게 MCP를 체험해보고 싶을 때
- UAF Python API가 공개되면 바로 쓰고 싶을 때

## runreal의 한계

- 전용 애니메이션 도구가 없음 (직접 Python 작성 필요)
- Blueprint 노드 그래프 편집 불가 (UE Python API 한계)
- 스크린샷 해상도 고정 (640x520)
- Python 코드에 주석 사용 불가 (Remote Execution 프로토콜 제한)
- 에디터가 켜져 있어야만 동작

## 핵심 의존성

```
@runreal/unreal-mcp (0.1.4)
├── @modelcontextprotocol/sdk (^1.12.1)  ← MCP 표준 SDK
├── unreal-remote-execution (^1.0.0)     ← UE 연결 라이브러리
└── zod (^3.25.48)                       ← 입력값 검증
```

외부 의존성이 3개뿐이라 매우 가볍습니다.
